🛒 Bazaar – E-Commerce Website

Bazaar is a fully functional and responsive E-Commerce Web Application developed as a major project. It provides users with a seamless online shopping experience, including browsing products, managing carts, and secure checkout functionality.

🚀 Features
🛍️ Browse products with detailed descriptions
🔍 Search and filter functionality
🛒 Add to cart / Remove from cart
💳 Checkout system (basic / integrated depending on your build)
👤 User authentication (Login / Signup) (if implemented)
📱 Fully responsive design (Mobile + Tablet + Desktop)
⚡ Fast and smooth UI/UX
📦 Organized product categories
