# 🎓 AIMERS – E-Learning Platform

## 📌 Overview

**AIMERS** is a full-stack E-Learning Platform built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**. It is designed to provide a seamless experience for both learners and instructors, similar to platforms like Udemy.

The platform allows users to **learn, teach, and grow** by accessing a wide range of courses with interactive features like video streaming, certifications, and progress tracking.

---

## 🚀 Features

### 👤 User Features

* 🔐 Secure Authentication (Login / Signup)
* 📚 Browse & Search Courses
* 🛒 Add to Cart & Purchase Courses
* 🎥 Video & Audio Streaming
* 📈 Track Learning Progress
* 🏆 Achievements & Certificates
* 💼 Job Opportunities Section

### 👨‍🏫 Instructor Features

* 📤 Upload Courses (Video, Audio, Notes)
* ✏️ Manage Course Content
* 📊 Track Student Engagement

### 🏢 Platform Features

* 🏠 Interactive Landing Page
* 🔎 Advanced Search Bar
* 🧾 User Dashboard
* 📦 Course Categories
* 📱 Fully Responsive Design

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```
git clone https://github.com/harshvardhan8922/aimers.git
cd aimers
```

### 2️⃣ Install Dependencies

#### For Backend:

```
cd server
npm install
```

#### For Frontend:

```
cd client
npm install
```

### 3️⃣ Run the Project

#### Start Backend:

```
npm start
```

#### Start Frontend:

```
npm start
```
